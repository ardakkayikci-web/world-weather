# 🌍 WorldWeather — Live Global Weather

A web-based real-time weather information system built with HTML, CSS, and JavaScript.

## 🔗 Live Demo
👉 https://ardakkayikci-web.github.io/world-weather/

## 📌 Features
- Search real-time weather for any city in the world
- Displays temperature, humidity, wind speed, pressure, and visibility
- Save favourite cities
- Fully responsive design

## 🛠️ Technologies Used
- HTML5, CSS3, JavaScript
- Open-Meteo Weather API
- Open-Meteo Geocoding API
- GitHub Pages (deployment)
